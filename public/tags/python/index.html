Structural Subtyping in Python

Structural subtyping in Python can be achieved using typing.Protocol and the @runtime_checkable decorator. Rather than employing abc.ABC for nominal subclasses, a cleaner approach involves combining @typing.runtime_checkable with typing.Protocol. By decorating a protocol class with @runtime_checkable, it becomes a runtime protocol which is amenable to isinstance() and issubclass(). Without @runtime_checkable, attempting isinstance() and issubclass() operations will result in a TypeError.
